Demonstrating the Box Model

The CSS box model is exxentially a box that wraps around every HTML element. It consists of: borders, padding, margins, and the actual content.

This text is the actual content of the box. We have added a 15px padding, 50px margin, and a 7px border. (and comic sans, for that extra touch.)
h3 {font-family: "Comic Sans MS"; padding: 15px; margin: 50px; border: 7px solid white;}

This is my H4. I have used a 4px "dashed" border.
h4 {border: 4px dashed white;}

This is my H5 inset. I've used a simple 4px inset.
h5 {border: 4px inset white;}
This is my H5 outset. I've used a simple 4px CSS based Outset.
.outset {border: 4px outset white;}

This is a normal P.
p {border: 4px solid white;}

This is a p, with the double.
.pdouble {border: 4px double white;}

This is a ridged p.
.pridge {border: 4px ridge white;}
